# ✈️ Airport Management System (C + MySQL)

## 📌 Overview

This project is a **console-based Airport Management System** developed in **C programming language** using **MySQL database**.

It simulates a real airport control system with multiple modules such as flights, passengers, crew, gates, security, notifications, and a live dashboard.

The goal of this project is to practice:
- C programming at a large scale
- Database integration with SQLite
- Modular software design
- Real-world system simulation

---

## 🚀 Features

### ✈️ Flight Management
- Add flights
- Delete flights
- Update flight status
- Display all flights

### 🛂 Passenger Check-in
- Register passengers
- Assign flight and seat
- Delete check-ins
- View passenger list

### 👨‍✈️ Crew Management
- Add crew members
- Assign crew to flights
- Delete crew members
- Display crew list

### 🚪 Gate Management
- Add gates
- Assign gate to flight
- Release gates
- View gate status

### 🛡️ Security System
- Add security records
- Check passenger authorization
- View security database

### 🔔 Notifications System
- Add alerts and messages
- Delete notifications
- View notifications history

### 📊 Dashboard
- Real-time overview of flights and gates
- SQL JOIN between flights and gates
- Monitoring interface
